# Sales Data Analysis

## Overview

This project analyzes sales data using Python.

## Features

* Basic statistics (sum, mean, max, min)
* Product line and country-wise sales
* Top-performing products
* Data visualization (bar & pie charts)
* Excel export

## Tools Used

* Pandas
* Matplotlib
* Seaborn

## How to Run

pip install pandas matplotlib seaborn openpyxl
python analysis.py
